# MarkSentence 
这是一个在听力中标记句子的工具，即精听工具，目前专门针对于`TOEFL`

- 可以下载TPO1-54的所有听力(包括**口语**和**写作**部分)
- 对音频截断, 并且做上标记, 生成标签
- 所做的标签会被保存, 在下次打开音频的时候会载入

## Usage
- 快捷键
	- 后退与快进：`←`, `→` （默认是3s）
	- 设置断点：`[` , `]`
	- 取消断点：`shift` + `[` , `]`
	- 保存当前标记： `Enter`
	- 播放：`F1`
