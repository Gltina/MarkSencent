Special for TOEFL
-------------------------------------------------------------------------------
这是一个在听力中标记句子的工具(**即精听工具**)，使用`QT5.9.0+Qt Creator 4.3.0`开发

功能
-------------------------------------------------------------------------------
- 可以下载TPO1-54的所有听力(包括**口语**和**写作**部分)
- 对音频截断，并且做上标记
- 所做的标签都可以保存下载，在下次听的时候还会载入

下载地址
-------------------------------------------------------------------------------
解压后双击 `"MarkSentence v.1.0.exe"` 即可

[点我下载](http://39.97.115.128:8000/SHATOEFL/MarkSentence.zip)
